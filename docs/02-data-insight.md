# 02 · Data Insight · "30 视频判断爆款"

> ⭐ 这是整个评分体系的**方法论起点**。所有评分维度、阈值、Phase 划分都源于这个洞察。

## 🎯 The Core Insight

> **一位 Creator 拍到 ~30 个视频左右，就能稳定判断他/她会不会出爆款。**

## 🔬 How This Insight Was Discovered

### 数据来源

不是用自家产品的数据（样本量太小、新人太多），而是**拉取了某头部竞品的几百位 Creator 的完整视频数据**——他们的 Creator 池更成熟、有完整生命周期数据。

涵盖维度：
- 每位 Creator 的视频总数
- 每条视频的发布时间、播放量、互动数据
- 视频内容类型（Trend / 正式模板 / Warm-up）

### 分析方法

把每位 Creator 的视频按时间排序，问几个问题：

1. **"出第一个 5K+/10K+/100K+ 爆款"** 发生在第几个视频？
2. **如果到第 N 个视频还没出爆款，未来出爆款的概率是多少？**
3. **早期表现（前 5 个视频）vs 后期表现（第 20+ 视频）有什么关系？**

### 关键发现

把所有 Creator 按"是否最终出过 10K+ 爆款"分两组，画累积视频数 vs 爆款率曲线，发现：

```
爆款率
  ▲
  │
  │              ┌──────────  (会爆款的 Creator)
  │           ┌──┘
  │       ┌───┘
  │   ┌───┘
  │   │
  │   │   ←──── 30 视频左右是关键拐点
  │───┘
  │
  ├─────────────────────────  (不会爆款的 Creator,
  │                              即使继续拍也几乎不出爆款)
  └──────────────────────────►
   0    10   20  ★30   40   50   累积视频数
```

**结论**：超过 **~30 个有效视频** 还没出 5K+ 爆款的 Creator，未来出爆款的概率显著下降——继续投入边际回报很低。

## 💡 Why This Matters

这个洞察的价值不在于"30 这个数字"，而在于：

| 价值 | 说明 |
|------|------|
| **给出了可执行的清退阈值** | "拍了 30+ 视频还没 5K" 比"感觉不行了"清晰 10 倍 |
| **校准了 Phase 划分** | <10 视频 = Early Validation；10-20 = Mid；20+ = Potential Validation |
| **为评分模型的"负分扣减"提供依据** | scoring.py 里 `n_valid >= 20 and max_views < 10000 → -10` 就来自这条规律 |

## 🔗 From Insight to Method

基于这个洞察，提炼出**「三段踢人」方法论**：

| 段位 | 判定信号 | 推荐动作 |
|------|---------|---------|
| 🟢 第一段 | < 30 视频，有早期突破信号 | 重点培养 |
| 🟡 第二段 | 20-30 视频，无突破但有边缘信号 | 观察期 + 干预 |
| 🔴 第三段 | 30+ 视频，无任何突破信号 | 清退候选 |

→ 进一步落地为评分模型的 **8 维度评分 + 4 级分级**，详见 [03-scoring-model.md](03-scoring-model.md)
